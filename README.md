# Go with Docker

Repository ini adalah bagian dari sharing session bersama dengan Kongkow IT Pekanbaru membahas seputar Golang

## Requirements
* Golang >=v1.11
* Docker Engine >=18.09
* Docker-Compose 

## Apa yang ada disini?
* Example Go Code
* Example Dockerfile
* Docker Compose Configuration
